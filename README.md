# DevSecOps Demo

Laboratorio práctico DevSecOps desarrollado en Python.

## Componentes

- Flask
- UV
- Pytest
- Bandit

## Objetivo

Implementar controles de seguridad dentro del ciclo de desarrollo de software utilizando herramientas modernas de DevSecOps.

## Roadmap
### v0.1.0 - Fundamentos

* API REST de Flask
* Gestión de paquetes UV
* Pruebas automatizadas con Pytest
* Pruebas estáticas de seguridad de aplicaciones (SAST) con Bandit

### v0.2.0 - Seguridad de la cadena de suministro de software

* Análisis de composición de software (SCA) con pip-audit
* Lista de materiales de software (SBOM) con CycloneDX
* Documentación de hallazgos de seguridad

### v0.3.0 - Seguridad de contenedores

* Contenerización con Docker
* Mejores prácticas de seguridad de contenedores
* Compilaciones reproducibles

### v0.4.0 - Pruebas dinámicas de seguridad

* Integración con OWASP ZAP
* Escaneos DAST automatizados
* Generación de informes de seguridad

### v0.5.0 - Automatización de seguridad de CI/CD

* Pipeline de GitHub Actions
* Comprobaciones de seguridad automatizadas
* Puertas de calidad de seguridad
